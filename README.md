# Foundation-Coursework
This repository contains the practical work and supporting materials for the coursework:
"Enhancing Secure Data Exchange, Algorithm Analysis, and Database Normalization".

## Repository Overview

### Task1 – Secure Data Exchange
- Contains `notes.txt` explaining encoding formats (Base64, ASCII, Hex, URL encoding) and secure protocols (HTTPS, TLS, SMTP, REST APIs).
- Diagrams may show data flow and encoding/decoding processes.
- Optional scripts (Python) for Base64, Hex, and URL encoding can be included.

### Task2 – Classroom Seating Arrangement
- Contains `notes.txt` explaining the problem scenario, P vs NP analysis, brute-force approach, and heuristic approach.
- No diagrams are included, but notes fully explain the practical work.

### Task3 – College Club Membership Management
- `create_tables.sql` : Database schema (Student, Club, Membership tables)
- `insert_data.sql` : Sample data inserted into tables
- `queries.sql` : SQL queries demonstrating SELECT and JOIN operations
- ER Diagram (`ER-diagram.png`) showing entities, attributes, primary keys, and foreign keys
